## 🚀 上传步骤

### 方法一：直接拖拽上传（推荐）

1. **访问GitHub**
   - 打开浏览器，访问 https://github.com
   - 登录您的账户

2. **创建新仓库**
   - 点击右上角的"+"号
   - 选择"New repository"
   - 填写信息：
     - Repository name: `design-pattern-bookstore`
     - Description: `图书商城管理系统 - 使用Spring Boot和多种设计模式实现`
     - 选择Public或Private
     - 不要勾选"Initialize this repository with README"
   - 点击"Create repository"

3. **上传文件**
   - 在新创建的仓库页面，点击"uploading an existing file"
   - 打开文件管理器，找到`upload-ready`文件夹
   - 将整个`upload-ready`文件夹拖拽到上传区域
   - 添加提交信息："初始提交：图书商城管理系统实现多种设计模式"
   - 点击"Commit changes"

### 方法二：压缩后上传

如果直接拖拽上传遇到问题：

1. **压缩文件夹**
   - 右键点击`upload-ready`文件夹
   - 选择"发送到" -> "压缩(zipped)文件夹"
   - 生成`upload-ready.zip`文件

2. **上传压缩文件**
   - 在GitHub仓库页面点击"uploading an existing file"
   - 选择`upload-ready.zip`文件
   - 添加提交信息："初始提交：图书商城管理系统"
   - 点击"Commit changes"

3. **解压说明**
   - 在README.md中添加说明，告知用户需要解压

## 📞 需要帮助？

如果在上传过程中遇到问题：

1. **文件太大**

   - 确保网络连接稳定

3. **权限问题**
   - 确认您有仓库的写入权限
   - 检查GitHub账户设置
